# Documentation Index

この `docs/` にはプロジェクトの設計書、ガイド、移行手順、リファレンスなどが含まれます。

主要フォルダ:

- [guides](guides/README.md) — 実装ガイド・チュートリアル
- [reference](reference/README.md) — 参照用ドキュメント（SPELL_* 等）
- [migration](migration/README.md) — 移行ガイド
- [backups](backups/README.md) — 移行中に退避した `.bak` ファイル
- [api](api) — API ドキュメント
- [architecture](architecture) — アーキテクチャ関連資料
- [dev](dev) — 開発ワークフローや手順

必要なら各 README を拡充して見出しや短い要約を追加してください。
